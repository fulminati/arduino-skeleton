# arduino-skeleton

## Get Started

### Create your project
```
$ arduinodk clone https://github.com/fulminati/arduino-skeleton MyArduinoProject
```

### Verify operation
Verify source code of your project
```
$ cd MyArduinoProject
$ arduinodk verify
```

### Upload operation
Upload source code in your board
```
$ cd MyArduinoProject
$ arduinodk upload
```
